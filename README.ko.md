# risk-radar

> 🇺🇸 [English README](./README.md)

**능동적 리스크 예측·모니터링 엔진. 프로젝트·사업·투자·조직 리스크를 사전 식별하고 확률×임팩트 매트릭스로 구조화.**

## 목표
반응형 리스크 관리에서 능동형 리스크 인텔리전스로 전환합니다. 포트폴리오 전체의 신흥 리스크를 체계적으로 식별하고, 잠재적 영향을 정량화하며, 위기가 되기 전에 경고 신호를 감지하는 지속적 모니터링을 수립합니다.

## 주요 기능
- **다중 도메인 리스크 분류**: 프로젝트, 사업, 투자, 조직 리스크 프레임
- **사전 식별 프로토콜**: 도메인별 패턴을 활용한 잠재 리스크 발굴
- **확률×임팩트 매트릭스**: 정량화된 리스크 점수 산정 및 심각도별 우선순위 지정
- **경고신호 시스템**: 각 리스크별 선행지표 정의 및 트리거 임계값 설정
- **주기적 스캔 프로토콜**: 자동화된 리스크 평가 사이클, 추이분석, 에스컬레이션 규칙
